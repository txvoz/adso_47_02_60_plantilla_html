Lista de etiquetas

Nombre Estructura Descripcion Propiedades Ejemplo
Img - Imagen < img src="[va la fuente de la imagen]" /> Esta etiqueta esta diseñada para implementar en el documento web un marco para montar, un recurso visual tipo imagen con extension png, jpg, jpeg y otros.
SRC: Atributo para poner la fuente del recurso
WITH: En esta propiedad se debe establecer el ancho del marco para la imagen.
Audio < audio controls src="[va la fuente del audio]" /> Esta etiqueta esta diseñada para implementar en el documento web un marco para montar, un recurso de audio con extension mp3.
SRC: Atributo para poner la fuente del recurso
controls: Atributo para ver el diseño visual del elemento en el navegador
Input type[text] < input type="text" /> Esta etiqueta tiene la funcionalidad de servir como entrada de datos de tipo de texto en un form
SRC: Atributo para poner la fuente del recurso
controls: Atributo para ver el diseño visual del elemento en el navegador
